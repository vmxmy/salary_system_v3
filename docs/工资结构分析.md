### 一、 总体工资概览 (Overall Salary Summary)

这类字段提供了工资总额和最终实得的宏观数据。

| 字段名称 | 描述 | 适用人员 |
| :--- | :--- | :--- |
| `应发工资` | **Gross Pay**: 员工在扣除任何费用（如税、社保）之前的全部应得收入。 | 正编、聘用 |
| `实发工资` | **Net Pay**: 员工在扣除所有应缴费用后，最终实际收到的金额。 | 正编、聘用 |
| `扣发合计` | **Total Deductions**: 从应发工资中扣除的所有项目的总和。 | 正编、聘用 |

---

### 二、 应发项目 (Earnings Components)

这是构成`应发工资`的各个具体项目，可以根据人员身份（正编 vs. 聘用）和工资体系进行细分。

#### 1. 正编人员核心工资 (Core Salary for Regular Staff)

这部分是正编人员（公务员、参公、事业编）工资结构的基础，体现了国家统一的工资制度。

| 字段名称 | 描述 | 适用人员 |
| :--- | :--- | :--- |
| `职务/技术等级工资` | **Position/Technical Grade Salary**: 根据员工的行政职务或技术职称等级确定的工资部分。 | 正编 |
| `级别/岗位级别工资` | **Level/Post Level Salary**: 根据员工的行政级别（如一级主任科员）或岗位级别确定的工资。 | 正编 |
| `岗位工资` | **Post Salary**: 主要针对事业单位人员，根据其所在岗位的职责和要求确定的工资。 | 事业编 |
| `薪级工资` | **Step Salary**: 主要针对事业单位人员，根据其工作年限和资历确定的工资，体现了“论资排辈”的特点。 | 事业编 |
| `见习试用期工资` | **Probationary Period Salary**: 针对新入职处于试用期或见习期的事业编人员的工资标准。 | 事业编 |

#### 2. 聘用人员核心工资 (Core Salary for Hired Staff)

聘用人员的工资结构相对更市场化和灵活。

| 字段名称 | 描述 | 适用人员 |
| :--- | :--- | :--- |
| `基本工资` | **Base Salary**: 聘用人员工资的基础部分，通常是固定的。 | 聘用 |
| `岗位工资` | **Post Salary**: 根据聘用人员所在岗位的价值和责任确定的工资部分。 | 聘用 |
| `固定薪酬 全年应发数` | **Fixed Salary (Annual Gross)**: 针对特定岗位（如专技人员）的全年固定薪酬总额。 | 聘用 |

#### 3. 绩效与奖金 (Performance & Bonuses)

这部分收入与员工的工作表现和考核结果直接挂钩。

| 字段名称 | 描述 | 适用人员 |
| :--- | :--- | :--- |
| `基础绩效奖` | **Basic Performance Bonus**: 正编人员的绩效部分，通常与月度或年度考核挂钩。 | 正编 |
| `月基础绩效` | **Monthly Basic Performance**: 事业编人员按月发放的基础绩效部分。 | 事业编 |
| `月奖励绩效` | **Monthly Bonus Performance**: 事业编人员根据月度表现额外获得的奖励性绩效。 | 事业编 |
| `绩效工资` | **Performance Salary**: 聘用人员工资中与绩效相关的浮动部分。 | 聘用 |
| `基础绩效` | **Basic Performance**: 聘用人员的基础绩效部分。 | 聘用 |
| `季度绩效考核薪酬` | **Quarterly Performance Salary**: 聘用人员根据季度考核结果发放的薪酬。 | 聘用 |

#### 4. 津贴与补贴 (Allowances & Subsidies)

为补偿员工在特殊岗位、特殊地区或特定生活成本下的支出而设立。

| 字段名称 | 描述 | 适用人员 |
| :--- | :--- | :--- |
| `公务员规范性津贴补贴` | **Civil Servant Standardized Allowances**: 国家为规范公务员收入而设立的统一津贴和补贴。 | 正编 |
| `公务交通补贴` | **Official Transportation Subsidy**: 为补偿公务员因公产生的交通费用而发放的补贴。 | 正编 |
| `93年工改保留补贴` | **1993 Salary Reform Retention Allowance**: 1993年工资改革时设立的一项历史保留补贴。 | 正编 |
| `独生子女父母奖励金` | **Single-Child Parent Bonus**: 对独生子女父母发放的奖励性津贴。 | 正编 |
| `信访工作人员岗位津贴` | **Petition-Handling Staff Post Allowance**: 对从事信访接待工作的员工发放的特殊岗位津贴。 | 正编、聘用 |
| `乡镇工作补贴` | **Township Work Subsidy**: 对在乡镇地区工作的员工发放的地区性补贴。 | 正编 |
| `岗位职务补贴` | **Post Duty Allowance**: 根据岗位职责发放的额外补贴。 | 正编 |
| `补助` | **Subsidy**: 聘用人员工资中的通用性补助项目。 | 聘用 |
| `津贴` | **Allowance**: 聘用人员工资中的通用性津贴项目。 | 聘用 |

#### 5. 补发与补扣 (Retroactive Adjustments)

用于处理因调薪、考核、社保基数变更等原因产生的工资追溯调整。

| 字段名称 | 描述 | 适用人员 |
| :--- | :--- | :--- |
| `一次性补扣发` | **One-time Retroactive Adjustment**: 非经常性的、一次性的工资补发或补扣。 | 正编、聘用 |
| `绩效奖金补扣发` | **Performance Bonus Retroactive Adjustment**: 针对绩效奖金的追溯调整。 | 正编、聘用 |
| `奖励绩效补扣发` | **Bonus Performance Retroactive Adjustment**: 针对奖励性绩效的追溯调整。 | 正编、聘用 |

### 三、 扣发项目 (Deductions)

从`应发工资`中扣除的法定和约定项目。

#### 1. 五险一金 (Social Insurance & Housing Fund)

| 字段名称 | 描述 | 适用人员 |
| :--- | :--- | :--- |
| `个人缴养老保险费` | **Personal Pension Contribution**: 员工个人缴纳的基本养老保险费用。 | 正编、聘用 |
| `个人缴医疗保险费` | **Personal Medical Insurance Contribution**: 员工个人缴纳的基本医疗保险费用。 | 正编、聘用 |
| `个人缴失业保险费` | **Personal Unemployment Insurance Contribution**: 员工个人缴纳的失业保险费用。 | 正编、聘用 |
| `个人缴住房公积金` | **Personal Housing Provident Fund Contribution**: 员工个人缴纳的住房公积金。 | 正编、聘用 |
| `个人缴职业年金` | **Personal Occupational Annuity Contribution**: 主要针对机关事业单位人员，在基本养老保险之外的补充养老保险。 | 正编 |

#### 2. 税务 (Taxation)

| 字段名称 | 描述 | 适用人员 |
| :--- | :--- | :--- |
| `个人所得税` | **Personal Income Tax**: 根据国家税法规定，从员工工资中代扣代缴的个人所得税。 | 正编、聘用 |

#### 3. 其他扣款 (Other Deductions)

| 字段名称 | 描述 | 适用人员 |
| :--- | :--- | :--- |
| `补扣社保` | **Retroactive Social Security Deduction**: 因社保缴费基数调整等原因，需要追溯补扣的社保费用。 | 正编、聘用 |
| `补扣（退）款` | **Deduction/Refund**: 通用性的补扣或退款项目。 | 聘用 |

### 四、 人员信息字段 (Personnel Information)

用于标识员工身份和基本信息的字段。

| 字段名称 | 描述 |
| :--- | :--- |
| `序号`, `人员编号`, `人员姓名`, `身份证`, `部门` | 员工的基本身份和组织信息。 |
| `人员身份` | 区分员工的管理类别，如 `已登记公务员`, `参公人员`, `事业技术工人`, `综合类` (聘用) 等。 |
| `人员职级` | 员工的行政或技术级别，如 `县处级正职`, `一级主任科员`, `综合四级` (聘用) 等。 |
| `工资统发`, `财政供养` | 标识工资发放方式和资金来源的标志位。 |
